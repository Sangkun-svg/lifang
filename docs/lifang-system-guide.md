# LIFANG INC 관리 시스템 사용 가이드

이 문서는 LIFANG INC 관리 시스템을 사용하는 관리자와 고객사 사용자를 위한 화면 사용 가이드입니다.

## 1. 사용 전 안내

- 본 시스템은 PC 화면 기준으로 제공됩니다.
- 모바일 화면은 별도로 지원하지 않습니다.
- 화면 폭이 좁을 경우 모바일 화면으로 바뀌지 않고, 일부 영역이 잘리거나 테이블 내부가 스크롤될 수 있습니다.
- 로그인 계정은 관리자 계정과 고객사 사용자 계정으로 구분됩니다.
- 관리자 계정은 관리자 페이지에만 접근할 수 있습니다.
- 고객사 사용자 계정은 사용자 페이지에만 접근할 수 있습니다.

## 2. 계정 종류

### 관리자

관리자는 전체 고객사 계정, 업로드 시트, 고객 요청을 관리합니다.

관리자가 할 수 있는 일:

- 고객사 계정 생성
- 고객사 정보 수정
- 고객사 삭제
- 엑셀 시트 업로드
- 업로드된 시트 데이터 확인
- 고객사별 시트 연결
- 최근 요청 확인
- 요청 상세 처리

### 고객사 사용자

고객사 사용자는 본인에게 연결된 시트 데이터를 조회하고, 필요한 차단 요청을 보낼 수 있습니다.

고객사 사용자가 할 수 있는 일:

- 본인 대시보드 확인
- 본인 내역 목록 조회
- 내역 상세 확인
- 차단 요청 생성
- 날짜, 진행상황, 정렬 조건으로 데이터 확인

## 3. 관리자 로그인

1. 관리자 로그인 페이지에 접속합니다.
2. 이메일과 비밀번호를 입력합니다.
3. `로그인` 버튼을 클릭합니다.
4. 로그인에 성공하면 회원목록 화면으로 이동합니다.

로그인 실패 시:

- 이메일 또는 비밀번호가 잘못되면 오류 메시지가 표시됩니다.
- 일반 사용자 계정으로 관리자 로그인을 시도하면 관리자 권한이 없다는 안내가 표시됩니다.
- 로그인 중에는 버튼에 로딩 상태가 표시됩니다.

## 4. 관리자 화면 구성

관리자 화면의 왼쪽 사이드바에는 주요 메뉴가 표시됩니다.

- `회원목록`: 고객사 계정 조회, 생성, 수정, 삭제
- `최근 요청`: 고객사 사용자가 보낸 요청 확인 및 처리
- `업로드 시트`: 엑셀 시트 업로드, 시트 목록 확인, 시트별 데이터 조회

업로드된 시트는 사이드바의 업로드 시트 영역에도 표시됩니다. 시트명을 선택하면 해당 시트의 데이터 목록으로 이동합니다.

## 5. 회원목록

회원목록에서는 등록된 고객사 계정을 테이블로 확인할 수 있습니다.

표시 정보:

- 가입 일자
- 업체명
- 담당자
- 이메일
- 연결 시트
- 상세정보

### 회원 검색

1. `업체명` 검색창에 업체명을 입력합니다.
2. 입력한 값이 포함된 회원만 목록에 표시됩니다.

### 날짜 검색

1. 검색 시작 날짜를 선택합니다.
2. 검색 종료 날짜를 선택합니다.
3. 선택한 기간에 해당하는 회원만 표시됩니다.

### 상세보기

회원 행의 `보기`를 클릭하면 회원 상세/수정 화면으로 이동합니다.

## 6. 신규 계정 생성

1. 회원목록 우측 상단의 `신규 계정 생성하기` 버튼을 클릭합니다.
2. 고객사 정보를 입력합니다.
3. 필요한 경우 연결할 구글시트를 선택합니다.
4. `계정 생성하기` 버튼을 클릭합니다.

입력 항목:

- 이메일
- 비밀번호
- 비밀번호 확인
- 업체명
- 담당자
- 구글시트

구글시트 선택:

- 시트명 검색창에 업로드된 시트명을 입력하면 선택 가능한 시트가 표시됩니다.
- 선택 가능한 시트가 없으면 `선택 가능한 시트가 없습니다.`가 표시됩니다.
- 시트는 필수값이 아니므로 선택하지 않아도 계정을 생성할 수 있습니다.
- 이미 다른 고객사에 연결된 시트는 선택할 수 없습니다.

비밀번호:

- 비밀번호 입력창 오른쪽의 아이콘을 누르면 입력한 비밀번호를 확인할 수 있습니다.
- 비밀번호와 비밀번호 확인 값이 일치해야 합니다.

생성 완료 후:

- 회원목록 화면으로 이동합니다.
- 생성한 고객사 계정이 목록에 표시됩니다.

## 7. 회원 수정

1. 회원목록에서 수정할 회원의 `보기`를 클릭합니다.
2. 업체명, 담당자, 연결 시트를 수정합니다.
3. `수정하기` 버튼을 클릭합니다.

수정 완료 후:

- 회원목록 화면으로 이동합니다.
- 변경된 정보가 목록에 반영됩니다.

주의:

- 이메일은 계정 식별 정보로 사용되므로 수정 화면에서 변경하지 않습니다.
- 연결 시트를 제거하면 해당 고객사는 더 이상 해당 시트 데이터를 볼 수 없습니다.

## 8. 회원 삭제

1. 회원 수정 화면에서 `삭제하기` 버튼을 클릭합니다.
2. 삭제 확인 안내를 확인합니다.
3. 삭제를 확정하면 회원과 연결 데이터가 삭제됩니다.

삭제 시 처리되는 데이터:

- 고객사 계정
- 고객사 사용자 정보
- 연결된 시트
- 연결된 시트 데이터
- 고객사가 보낸 요청 데이터

주의:

- 삭제는 복구가 어려운 작업입니다.
- 삭제 전 반드시 대상 고객사와 연결 시트를 확인해야 합니다.

## 9. 업로드 시트

업로드 시트 화면에서는 엑셀 파일을 업로드하고, 업로드된 시트 목록을 관리합니다.

### 시트 업로드

1. 관리자 사이드바에서 `업로드 시트`를 클릭합니다.
2. `시트명`을 입력합니다.
3. 엑셀 파일을 선택합니다.
4. `시트 업로드` 버튼을 클릭합니다.

업로드 중에는 버튼에 로딩 상태가 표시됩니다.
시트명을 입력하지 않으면 선택한 엑셀 파일명이 시트명으로 자동 적용됩니다.

업로드 완료 후:

- 업로드된 시트 목록에 새 시트가 표시됩니다.
- 사이드바의 업로드 시트 목록에도 시트가 표시됩니다.
- 시트명을 클릭하면 시트 데이터 목록으로 이동합니다.

중복 안내:

- 같은 원본 파일명의 미매칭 시트가 있으면 새로 업로드할지 확인 안내가 표시될 수 있습니다.

### 시트 목록 확인

시트 목록에서는 다음 정보를 확인할 수 있습니다.

- 시트명
- 원본 파일명
- 데이터 개수
- 매칭 상태
- 삭제 예정 상태
- 삭제 버튼

매칭되지 않은 시트는 일정 기간 후 자동 삭제 대상이 될 수 있습니다. 화면의 삭제 예정 안내를 확인하세요.

### 시트 삭제

1. 업로드 시트 목록에서 삭제할 시트의 `삭제하기` 버튼을 클릭합니다.
2. 삭제 확인 안내를 확인합니다.
3. 삭제를 확정합니다.

삭제 가능한 경우:

- 아직 어떤 고객사에도 연결되지 않은 시트

삭제 불가능한 경우:

- 특정 고객사에 이미 연결된 시트

삭제 불가능한 시트를 삭제하려고 하면 어떤 고객사에 등록되어 있는지 안내가 표시됩니다.

## 10. 시트 데이터 목록

업로드된 시트를 선택하면 해당 시트의 데이터 목록을 확인할 수 있습니다.

표시 정보:

- 상품 이미지
- 상품명
- 링크
- 플랫폼 또는 카테고리
- 가격
- 판매수량
- 진행 상태
- 생성일 또는 검색일

### 날짜 필터

1. 검색 시작 날짜를 선택합니다.
2. 검색 종료 날짜를 선택합니다.
3. `조회` 버튼을 클릭합니다.

기본 날짜는 해당 시트 데이터의 날짜 범위를 기준으로 자동 설정됩니다.

### 정렬

정렬 조건을 선택해 목록을 정렬할 수 있습니다.

- 판매가
- 판매수량
- 플랫폼

### 상세보기

목록에서 개별 데이터를 선택하면 시트 데이터 상세 화면으로 이동합니다.

## 11. 시트 데이터 상세

시트 데이터 상세 화면에서는 특정 상품의 상세 정보와 처리 정보를 확인하고 수정할 수 있습니다.

확인 가능한 정보:

- 시트명
- 검색 날짜
- 매칭 상태
- 침해 플랫폼
- 침해 제품명
- 침해 업체명
- 판매가
- 판매수량
- 판매링크

수정 가능한 처리 정보:

- 진행 상황
- 차단 신고
- 차단 신고 승인
- 차단 신고 미승인 사유
- 상대방 이의신청
- 이의신청 승인
- 이의 신청
- 차단신고 재신고
- 차단신고 재신고 승인
- 차단신고 재신고 미승인 사유

드롭다운에서 기본값을 선택할 수 있고, `직접입력`을 선택하면 직접 값을 입력할 수 있습니다.

수정 후 `저장하기` 버튼을 클릭하면 변경사항이 저장됩니다.

## 12. 최근 요청

최근 요청 화면에서는 고객사 사용자가 보낸 요청을 확인할 수 있습니다.

표시 정보:

- 요청 일시
- 요청자
- 상품명
- 플랫폼
- 처리 상태
- 상세정보

정렬 조건:

- 날짜순 최신
- 날짜순 오래된
- 플랫폼순
- 유저순
- 이름순
- 상태순

요청이 없으면 `아직 접수된 요청이 없습니다.`가 표시됩니다.

## 13. 최근 요청 상세

최근 요청 상세 화면에서는 고객사 사용자가 요청한 상품 정보와 처리 상태를 수정할 수 있습니다.

확인 가능한 정보:

- 요청자
- 요청 구분
- 처리 상태
- 검색 날짜
- 시트 데이터
- 상품 정보
- 침해 제품명
- 침해 업체명
- 침해 플랫폼
- 판매가
- 판매수량
- 판매링크

수정 가능한 처리 정보:

- 처리 상태
- 차단 여부
- 차단 신고
- 차단 신고 승인
- 차단 신고 미승인 사유
- 상대방 이의신청
- 이의신청 승인
- 이의 신청
- 차단신고 재신고
- 차단신고 재신고 승인
- 차단신고 재신고 미승인 사유

드롭다운에서 값을 선택하거나, `직접입력`을 선택해 직접 입력할 수 있습니다.

수정 후 `저장하기` 버튼을 클릭하면 변경사항이 저장됩니다.

## 14. 사용자 로그인

1. 사용자 로그인 페이지에 접속합니다.
2. 이메일과 비밀번호를 입력합니다.
3. `로그인` 버튼을 클릭합니다.
4. 로그인에 성공하면 대시보드로 이동합니다.

로그인 실패 시:

- 이메일 또는 비밀번호가 잘못되면 오류 메시지가 표시됩니다.
- 관리자 계정으로 사용자 로그인을 시도하면 관리자 로그인 페이지로 이동 안내가 표시됩니다.
- 로그인 중에는 버튼에 로딩 상태가 표시됩니다.

## 15. 사용자 대시보드

대시보드에서는 고객사 본인에게 연결된 시트 데이터를 통계로 확인할 수 있습니다.

상단 필터:

- 조회 상품
- 검색 시작 월
- 검색 종료 월
- 조회 버튼

조회 상품을 변경하면 해당 상품 데이터 기준으로 날짜 범위와 통계가 다시 적용됩니다.

표시 영역:

- 총 검색 수
- 확정된 위조품 수
- 차단 신청한 위조품 수
- 차단 완료된 위조품 수
- 플랫폼 별 위조품 수
- 플랫폼 별 비율 차트
- 탑셀러
- 월별 위조품 현황

그래프에 마우스를 올리면 해당 항목의 상세 수치를 툴팁으로 확인할 수 있습니다.

데이터가 없으면 `표시할 데이터가 없습니다.`가 표시됩니다.

## 16. 사용자 내역목록

사용자는 본인에게 연결된 시트 데이터를 내역목록에서 확인할 수 있습니다.

표시 정보:

- 상품 이미지
- 상품명
- 링크
- 플랫폼
- 판매가
- 판매수량
- 진행 상태
- 차단 신청

필터와 정렬:

- 판매가
- 판매수량
- 플랫폼
- 전체
- 미신청
- 신고완료
- 차단완료
- 검색 날짜

날짜를 선택한 뒤 `조회` 버튼을 클릭하면 해당 기간 데이터만 표시됩니다.

페이지네이션:

- 목록 하단에서 페이지를 이동할 수 있습니다.
- 페이지 번호는 10개 단위로 표시됩니다.
- 첫 페이지, 이전 페이지, 다음 페이지, 마지막 페이지 버튼을 사용할 수 있습니다.

차단 신청:

1. 목록에서 차단 신청이 필요한 항목의 체크박스를 선택합니다.
2. 저장 중 로딩이 표시됩니다.
3. 요청이 저장되면 관리자 최근 요청에 해당 데이터가 표시됩니다.

## 17. 사용자 내역 상세

내역 상세 화면에서는 상품별 상세 정보를 확인할 수 있습니다.

확인 가능한 정보:

- 상품 이미지
- 침해 플랫폼
- 침해 제품명
- 침해 업체명
- 판매가
- 판매수량
- 판매링크
- 진행 상황
- 차단 신고 정보
- 이의신청 정보
- 재신고 정보

`목록으로` 버튼을 클릭하면 내역목록으로 돌아갑니다.

## 18. 로딩, 빈 상태, 오류 상태

시스템은 주요 작업 중 로딩 상태를 표시합니다.

로딩이 표시되는 경우:

- 로그인 중
- 계정 생성 중
- 회원 수정 중
- 회원 삭제 중
- 시트 업로드 중
- 시트 삭제 중
- 요청 저장 중
- 페이지 이동 중

데이터가 없을 때는 빈 상태 안내가 표시됩니다.

예시:

- 등록된 회원 데이터가 없습니다.
- 업로드된 시트가 없습니다.
- 표시할 내역 데이터가 없습니다.
- 아직 접수된 요청이 없습니다.

오류가 발생하면 화면에 사용자용 오류 메시지가 표시됩니다. 같은 오류가 반복되면 관리자에게 문의하세요.

## 19. 권한 안내

관리자와 고객사 사용자는 접근 가능한 화면이 다릅니다.

- 관리자 계정은 관리자 페이지에 접근할 수 있습니다.
- 고객사 사용자 계정은 사용자 페이지에 접근할 수 있습니다.
- 고객사 사용자는 본인에게 연결된 시트 데이터만 볼 수 있습니다.
- 권한이 없는 페이지에 접근하면 로그인 페이지로 이동하거나 권한 안내가 표시됩니다.

## 20. 자주 묻는 질문

### 로그인했는데 다른 로그인 페이지로 이동합니다.

계정 종류가 맞지 않을 수 있습니다. 관리자 계정은 관리자 로그인 페이지를, 고객사 사용자 계정은 사용자 로그인 페이지를 사용해야 합니다.

### 업로드한 시트가 고객사 화면에 보이지 않습니다.

업로드한 시트가 해당 고객사 계정에 연결되어 있는지 확인하세요. 회원 생성 또는 회원 수정 화면에서 시트를 선택해야 고객사 사용자가 볼 수 있습니다.

### 시트를 삭제할 수 없습니다.

이미 고객사에 등록된 시트는 삭제할 수 없습니다. 먼저 회원 수정 화면에서 해당 시트 연결을 해제해야 합니다.

### 내역목록에 데이터가 보이지 않습니다.

검색 날짜 범위나 진행상황 필터를 확인하세요. 필터 조건에 맞는 데이터가 없으면 목록이 비어 보일 수 있습니다.

### 차단 신청을 눌렀는데 요청이 보이지 않습니다.

저장 중 로딩이 끝났는지 확인하세요. 오류 메시지가 표시되면 다시 시도하거나 관리자에게 문의하세요.

### 그래프 수치가 예상과 다릅니다.

대시보드의 조회 상품과 검색 날짜 범위를 확인하세요. 조회 조건이 바뀌면 통계도 해당 조건 기준으로 다시 계산됩니다.

## 21. 사용 시 주의사항

- 회원 삭제와 시트 삭제는 복구가 어려우므로 신중하게 진행하세요.
- 시트 업로드 전 시트명과 원본 파일을 확인하세요.
- 고객사에 연결할 시트가 맞는지 확인한 뒤 계정을 생성하거나 수정하세요.
- 처리 정보 저장 후에는 상세 화면에서 변경사항이 반영되었는지 확인하세요.
- 화면이 좁아 테이블이 잘리면 테이블 영역을 가로로 스크롤해 확인하세요.
